# Day 6 - Topic 2: ES Modules (30 Tasks)

Each folder contains one or more related tasks with runnable .js files.
Run any index.js with Node.js, for example:

    node task21-22/index.js

Note: package.json sets "type": "module" so all .js files here use ES Module syntax.
