# Training Code

A structured learning repository for practicing modern web development fundamentals using HTML, CSS, Bootstrap, and JavaScript.

## Overview

This repository contains daily practice files, demo examples, and training tasks organized by topic and day. It is designed for students and beginners who want to strengthen their frontend development skills through hands-on exercises.

## Project Structure

- Backend/ - JavaScript practice files and daily coding exercises
- Frontend/ - Frontend learning materials and UI examples
- Frontend/HTML/ - HTML structure and webpage examples
- Frontend/BooTstrap/ - Bootstrap components and responsive design demos
- Frontend/JavaScript/ - JavaScript examples and interactive projects
- Traning TASK/ - Assignments and class tasks for practice

## Technologies Covered

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Responsive web design
- Basic frontend interactivity

## Getting Started

1. Clone or open this repository in your preferred code editor.
2. Navigate to any lesson or task folder.
3. Open the relevant HTML file in a browser to view the output.

### Example

- Open [Frontend/HTML/demo1.html](Frontend/HTML/demo1.html)
- Or open [Backend/Day 1/demo.html](Backend/Day%201/demo.html)

## How to Use

- Start with the demo files to understand the concept.
- Practice the task files after completing the examples.
- Modify the code to experiment and build your own versions.

## Learning Objectives

- Understand basic HTML and CSS structure
- Practice responsive design with Bootstrap
- Learn JavaScript fundamentals
- Improve problem-solving through daily coding tasks
